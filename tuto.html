Les données :
Où et comment se procurer un jeu de données ?

Afin de réaliser une carte interactive, il est possible de se procurer des jeux de données sur divers sites dédiés. Pour cette carte, c’est le site de data.gouv.fr qui a été choisit.

Qu’est-ce que data.gouv.fr ? « une plateforme de diffusion de données publiques de l'État français. data.gouv.fr est développé par Etalab, une mission placée sous l'autorité du Premier ministre. » (source : Wikipédia)

data.gouv.fr se présente donc comme une plateforme ouverte de données publiques du Gouvernement français. Ainsi, sur cette plateforme, nous pouvons retrouver des jeux de données, comprenant un ensemble cohérent de ressources ou d’informations (fichiers de données, fichiers d’explications, API, lien…) et de métadonnées (présentation, date de publication, mots-clefs, couverture géographique/temporelle…), sur un thème donné. Avec ces jeux de données, il est possible de :

  1. les consulter et télécharger sous divers formats,
  2. les réutiliser à des fins autres que celles pour lesquelles elles ont été produites ou reçues,
  4. contribuer en, en ajoutant de nouveaux, ou encore en créant des visualisations.

Pour se procurer un jeu de données sur data.gouv.fr, il existe deux moyens de recherche :

  1. une recherche par thème,
  2. Une recherche par mot-clef.

Ici, c’est tout simplement une recherche par mot-clef qui a été effectué, avec le mot-clef : « coronavirus ».
