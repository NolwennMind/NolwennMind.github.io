L'emoji

Émoji est un terme issu du japonais pour désigner les pictogrammes utilisés dans les messages électroniques et les pages web japonaises, qui se sont répandus dans le monde entier. Le mot emoji signifie littéralement « image » + « lettre » ; la ressemblance avec « émotion » est un jeu de mot interculturel.
